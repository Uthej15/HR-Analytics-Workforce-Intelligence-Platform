# HR Analytics & Workforce Intelligence Platform

## Project Overview
A comprehensive HR analytics and workforce intelligence solution built using SQL, Power BI, and Python to analyse employee attrition, workforce demographics, salary trends, productivity insights, and executive KPIs.

---

## Tech Stack
- SQL
- MySQL
- Power BI
- DAX
- Python
- Pandas

---

## Key Features
- Employee Attrition Analysis
- Workforce KPI Monitoring
- Salary & Performance Insights
- Workforce Demographics Analysis
- Interactive Executive Dashboards
- Advanced SQL Reporting
- Workforce Productivity Tracking
